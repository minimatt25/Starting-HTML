How do things work?

this is a paragraph (using <p>), you can't actually use greater than and less than symbols as they are reserved characters.
instead i used & lt and & gt.

  1. the cool line splitting the screen at the top you can use <hr>.
  2. to change the background colour, inside <body> add a dash and put style="background-color:"
  3. to change text colour go inside the <p> or <h> and do - style="color"
  4. to change font, go inside and use font-family
  5. to make an ordered list like this use <ol>, then on the inside use <li> for each number.
if you need to change a multiple things at the same time about some text (e.g. font and colour),
you can put <style> in the header this can change stuff for every instance of the text, like how all my h1's have colour and font
